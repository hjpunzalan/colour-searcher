# Colour Searcher

_Dependencies_

- [node](https://nodejs.org/en/)

### **Initial Setup**

Run the following when setting up the project for the first time:

```
npm run install
```

### **Daily Development**

Start project

```
npm run dev
```

Git hook Husky will run eslint and prettier per commit

## Stack

- Vite.js with React

## Style

- Valera Round - https://fonts.google.com/specimen/Varela+Round
